# 3X-UI Sub Templates

[English](README.md) | **Русский**

Коллекция кастомных шаблонов страницы подписки для VPN-панели [3X-UI](https://github.com/MHSanaei/3x-ui).
Каждый шаблон — современная страница «всё в одном файле», которая заменяет стандартную страницу подписки и даёт вашим пользователям удобный личный кабинет.

## Общие возможности

Все шаблоны коллекции построены на одних принципах:

- **Один самодостаточный файл** — все стили, скрипты и иконки встроены. Никаких внешних картинок и CDN, поэтому шаблоны работают «из коробки» с Go-сервером, который отдаёт страницы подписки 3X-UI.
- **Кнопки копирования подписки** — копирование ссылок Xray / Sing-box и Clash / Mihomo в один клик с видимой реакцией «Скопировано!».
- **Карточки конфигураций** — каждая ссылка разбирается на клиенте и отображается с цветными тегами протокола / транспорта / шифрования (VLESS, VMESS, Trojan, Shadowsocks, Hysteria2, TUIC, Reality, WS, gRPC и другие).
- **Умная полоска трафика** — цвет плавно меняется от зелёного к красному по мере расхода трафика; для безлимитных тарифов полоса остаётся полностью зелёной.
- **Анимированное ЧаВо** — раскрывающиеся строки вопросов с плавной анимацией и автопрокруткой к открытому ответу.
- **Блок контактов поддержки** — ссылка на поддержку и кнопки Telegram / WhatsApp / GitHub, подсвечивающиеся фирменными цветами при наведении.
- **Адаптивность** — корректно выглядит на телефоне, планшете и десктопе.
- **Простая настройка** — в начале каждого файла есть карта настраиваемых участков с точными диапазонами строк: логотип, цвета, ЧаВо, контакты и прочее.

## Быстрый старт

1. Выберите шаблон из таблицы выше и возьмите его `sub.html`.
2. Замените им шаблон страницы подписки вашей панели 3X-UI.
3. Откройте файл и следуйте **карте настраиваемых участков** в комментарии в самом начале:
   - замените плейсхолдер `Your Logo` на свой логотип,
   - укажите свои ссылки Telegram / WhatsApp / GitHub (по умолчанию стоит `#`),
   - при желании отредактируйте ответы ЧаВо и цвета.
4. Готово — панель отрендерит страницу с данными вашей подписки.

## Переменные шаблона

Страницы используют стандартные переменные Go-шаблона, которые передаёт 3X-UI:

`.subTitle`, `.enabled`, `.emails`, `.sId`, `.expire`, `.lastOnline`, `.used`, `.remained`, `.total`, `.download`, `.upload`, `.links`, `.subUrl`, `.subClashUrl`, `.subSupportUrl`

## Настройка

Всё, что обычно требуется менять, перечислено в комментарии в начале каждого шаблона с точными диапазонами строк:

- заголовок вкладки и цветовая тема (CSS-переменные в `:root`),
- логотип в шапке и заголовок страницы,
- кнопки подписки Xray / Clash (названия, подписи, SVG-иконки),
- контакты поддержки и соцкнопки,
- секция ЧаВо — новые вопросы добавляются копированием блока `<details class="faq-item">`,
- длительность анимации ЧаВо (`FAQ_ANIM_MS`).
